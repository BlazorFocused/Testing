# Documentation

Contains developer documentation and content hosted in GitHub Pages

## Requirements

- [DocFx](https://dotnet.github.io/docfx/)

## Install

`dotnet tool update -g docfx`

## General Build

`docfx path/to/docfx.json`

## Generate Metadata Only

`docfx metadata`

## Build/Serve Local Web

`docfx build --serve`
